Crawl

This is a very easy level.A grid is displayed to you in which all the cells except one are alike.Find that odd cell to gain points.Size of the grid and complexity increases with lapse of time. If you are unable to find that odd cell,do not panic.Use the hint provided.

Walk

This is a difficult level.A grid is displayed to you in which all the cells except one are alike.The odd cell keeps flying all over the grid.Try to catch that cell to gain points. Be quick to find as there is no hint provided.

Race

This is a tricky game.You will be shown a color before you start playing.A grid is displayed in which all the cells except one are alike.The odd cell is displayed in different colors, catch that cell which is of the color you are shown earlier.Do not catch that odd cell with other colors as that reduces your score.Be quick to catch as there is no hint provided.
